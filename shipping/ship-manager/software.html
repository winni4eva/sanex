Frequently Asked Questions
How do I import and/or export from the hold file?

If you’re a FedEx Ship Manager Software PassPort user, follow these steps:

  1. Select Process Transactions from the PassPort drop-down menu at the top of the screen.
  2. Select a processing mode on the PassPort screen.
  3. Select the input file as usual. An output file is optional.
  4. To import all transactions into the hold file, check Process.

To find out how to export from the hold file or to get other details, refer to this helpful job aid.

Why can't I see my shipping rates? How can I fix that?

FedEx Ship Manager Software does not display rates for bill recipient or bill third party.

FedEx Ship Manager Software does not support international list rates.

While FedEx Express and FedEx Ground rates, surcharges and discounts automatically download to your system, you may need to download them manually if your rate is showing as $0.00 on the dynamic rate preview or rate quote. Here's how to do a Demand Download:

1.    Select Download from the Utilities menu. The Demand Download screen displays.

2.    Select the appropriate checkboxes, as applicable.

  • For FedEx Express, select the Express Domestic List Rates checkbox, Express Domestic Discounts checkbox and Express International Rates checkbox, as applicable.
  • For FedEx Ground, select the Ground List Rates checkbox and Ground Discounts checkbox, as applicable.
  • For earned discounts, select the Express Earned Discounts checkbox and the Ground Earned Discounts checkbox, as applicable.
  • For FedEx Ground discounted rates (for FedEx® Transborder Distribution shipments for Canada-origin shippers), select the Ground Discounts checkbox.

3.    Click OK.

The service I'm looking for is not available in my Service Type drop-down menu.

The service is either not set up in your system or is not available in the area to where you’re shipping. For more information, see the FedEx Service Guide or contact your FedEx account executive.

What's the difference between FedEx Priority Overnight® and FedEx Standard Overnight®?

Find the answer to this and other service questions in the FedEx Service Guide.

Accessing, Upgrading, and Setting up FedEx Ship Manager Software
How can I receive FedEx Ship Manager Software?

You can receive FedEx Ship Manager Software in two ways:

  • Call the FedEx Ship Manager Technical Support Center at 1.877.FDX Assist 1.877.339.2774 and say “FedEx Ship Manager Software” when prompted.
  • Request access to the current version of FedEx Ship Manager Software by completing and submitting the FedEx® Technology Solutions email form:
    • Select a topic.Click FedEx Ship Manager Software from the drop-down menu.
    • Select a product or service. Select FedEx Ship Manager Software from the drop-down menu.
    • Select a category. Select Registration from the drop-down menu.
    • Tell us if you are requesting FedEx Ship Manager Software. Select Yes from the drop-down menu.
    • Comments. Type "Requesting FedEx Ship Manager Software."
    • Click Submit.
How do I upgrade my version of FedEx Ship Manager Software?

You can upgrade your FedEx Ship Manager Software in three ways:

  • Accept the automatic upgrade from FedEx.
  • Call the FedEx Ship Manager Technical Support Center at 1.877.FDX Assist 1.877.339.2774 and say “FedEx Ship Manager Software” when prompted.
  • Request access to the current version of FedEx Ship Manager Software by completing and submitting the FedEx® Technology Solutions email form:
    • Select a topic. Click FedEx Ship Manager Software from the drop-down menu.
    • Select a product or service. Select FedEx Ship Manager Software from the drop-down menu.
    • Select a category. Select Registration from the drop-down menu.
    • Tell us if you are requesting FedEx Ship Manager Software. Select Yes from the drop-down menu.
    • Comments. Type “Requesting FedEx Ship Manager Software.”
    • Click Submit.
How do I upgrade my version of FedEx Ship Manager hardware?

Upgrade your hardware version by accepting the automatic upgrade. To receive it sooner, call 1.877.FDX.Assist 1.877.339.2774 between 7 a.m.–9 p.m. CST Monday–Friday and 9 a.m.–3 p.m. CST Saturday. Request to have your FedEx Ship Manager hardware system upgraded to the current version.

Can there be more than one meter number per FedEx Ship Manager Software or FedEx Ship Manager hardware account number?

Yes. You can assign 50 meter numbers for each FedEx Ship Manager Software or FedEx Ship Manager hardware account number.

How do I update my sender information, including account number, address or telephone number?

Select Senders from the Databases menu at the top of the screen. When the View Senders screen displays, make your changes and click OK.

How can I set up recipient preferences in the Recipient Database?

To set up recipient preferences for third-party billing in the Recipient database, follow these steps:

  1. Select Recipient from the Databases menu. The View Address Book screen displays.
  2. For an existing recipient, select the recipient and click View/Edit. The View/Edit Recipient screen displays. For a new recipient, click Add. The Add Recipient screen displays.
  3. On either screen, click Recipient Preferences. The View/Edit Recipient screen for setting preferences displays.
  4. Click Domestic Field Preferences for U.S. and Canada domestic recipients. You can select International Field Preferences for international recipients, but a different set of preferences displays.
  5. Select FedEx Express Payment Type.
  6. Select Constant as the Behavior and choose Third Party from the Field Values drop-down menu.
  7. Click OK.
  8. On the View/Edit Recipient screen, click OK.
  9. On the View Address Book screen, click OK.

Note: Recipient preferences override sender preferences that have been selected in Shipping Profiles for the same field.

How can I add a record to the Sender Database?

The sender information section on the Shipment Details screen shows the current sender and allows you to change the sender and the sender’s return address.

To print the return address information on the shipping label, leave the default as current sender or a select a different return address from the Change Return Address menu.

To add a new record to the Sender Database, follow these steps:

  1. Select Sender from the Databases menu. The View Senders screen displays.
  2. Click Add. The Add Sender screen displays.
  3. Enter a new Sender ID (e.g., ABC) and the remaining information you want to save. Required fields appear in bold on the screen. The new Sender ID can contain up to 25 characters.
  4. Click OK to save the record and return to the View Senders screen.
  5. Click the new sender (ABC) in the Sender List that you want as the default sender for the return address and click Return Address to the right of the list.
  6. Click OK.
  7. In the Sender Information section of the Shipment Details screen, the default sender for the return address (e.g., ABC – John Smith) displays in the Change Return Address field.
  8. If someone changes the default sender for the return address and you need to change it back, select the ABC Return Sender option (e.g., ABC – John Smith) from the Change Return Address drop-down menu.

Shipping a Package (General)
What type of printer will I need?

To ship domestic, international and return shipments, you will need two types of printers: label printer and report printer. You can set up these printers by selecting Printer Setup from the Utilities menu at the top of the screen.

  • Label printer: Prints the shipping label for your shipments
  • Report printer: Prints your designated reports (e.g., Shipment, Shipment history, FedEx Ground Manifest)

FedEx provides both printers for qualified FedEx Ship Manager hardware customers. If you are a FedEx Ship Manager Software customer, your FedEx account executive can order a printer and peripherals for you through the FedEx Customer Choice Program.

Can I find out how much my shipment will cost?

FedEx Ship Manager Software and FedEx Ship Manager hardware allow you to get a rate estimate for a shipment before you send the shipment information to FedEx for processing. You can also display rate quotes automatically for all shipments when you enter minimum shipping information. Rate quotes appear at the bottom of the Shipment Details screen. Note: The rate estimate and actual shipping charge may vary.

To display rate quotes, follow these steps:

  1. Once you enter the necessary information about the shipment, the Rate Quote button displays on the bottom of the Shipment Details screen.
  2. If you are shipping a multiple-piece shipment, click Rate Multiple-Piece Shipment.
  3. Click OK.

To display rate quotes automatically, update your system settings following the instructions below.

  1. Select System Settings from the Customize menu.
  2. Select your system number and click Modify. Note: In most cases, only one system number appears. If you want to add system numbers, please contact your FedEx sales professional.
  3. Click the FedEx Customer Admin tab.
  4. Check the Enable Dynamic Rate Preview box in the Shipping Screen section.
  5. Click OK.

For FedEx Express® and FedEx Ground® shipments, your quote is based on rates associated with your FedEx account number. In either case, the rate you receive may be different from the actual charges for your shipment. Differences may be based on the actual weight, dimensions and other factors. For details on how shipping charges are calculated, consult the FedEx Service Guide.

 

How can I add my handling charges to the rate quote I receive from FedEx?

For your convenience, you can set up your FedEx Ship Manager Software or FedEx Ship Manager hardware account so that your handling charges are added to the rate quote you receive from FedEx. Handling charges are added to the actual shipping charges by the shipper. These are not charges assessed by FedEx and are not paid to FedEx.

To add or modify handling charges for this package, click Additional Handling and enter a fixed or variable amount.

To enable this option, follow these steps:

  1. Select Shipping Profiles from the Customize drop-down menu at the top of the screen.
  2. When the View Shipping Profiles screen displays, select a profile (e.g., Default Domestic Shipping Profile) and click View/Edit.
  3. When the View/Edit Domestic Shipping Profile screen displays, click the FedEx Express Preferences tab.
  4. Check the Include Additional Handling Charge option in the Handling Charge section. Enter a fixed or variable amount and click OK.
  5. Then click OK on the View Shipping Profiles screen. The additional handling charge option is now enabled.
How can I make FedEx® Standard List Rates the default for FedEx Express shipping labels?

To make FedEx Standard List Rates the default for FedEx Express shipping labels, follow these steps:

  1. Choose Shipping Profiles from the Customize menu at the top of the screen. The View Shipping Profiles screen displays.
  2. Select a profile (e.g., Default Domestic Shipping Profile) and click View/Edit. The View/Edit Domestic Shipping Profile screen displays.
  3. Click the FedEx Express Preferences tab at the top of the screen.
  4. In the Label Format section, select Customize Document tab configuration. Click Doc Tab Configuration. The Doc Tab Configuration screen displays.
  5. Select the Package Doc Tab Configuration tab. (This is usually the default tab.)
  6. In the last column on the right next to the Shipping Field (first row), use your up/down arrow and select List Net Charge.
  7. In the last column on the right next to the Total Field (fourth row), use your up/down arrow and select List Total Customer Charge.
  8. Click OK. The View/Edit Domestic Shipping Profile screen displays.
  9. Click OK. The View Shipping Profiles screen displays.
  10. Click OK. Your default is now FedEx Standard List Rates for FedEx Express® shipments.
What type of shipping label will I need?

Depending on the destination for the shipment you are sending, you will need to prepare either a domestic or international shipping label. With FedEx Ship Manager Software and FedEx Ship Manager hardware, you can prepare both of these online and print them using your label printer or laser printer.

Can I print a label for a shipment that will be sent tomorrow?

Yes. You can process a shipping label up to 10 days in advance for FedEx Express® shipments (5 days in advance for FedEx Express® Freight shipments).

U.S. Domestic Shipping
How do I ship a U.S. domestic package?

If you're shipping a package within the U.S. or Canada, click the Ship tab at the top of the screen. Use the resulting Shipment Details screen to complete the shipment information. FedEx Ship Manager Software and FedEx Ship Manager hardware guide you through the steps to complete this task. For details, refer to the User Guide. When all required information for the shipping label is complete, you can request a courtesy rate quote to preview shipping charges, cancel the shipment or generate the shipping label, which is printed from your browser on a laser printer.

What delivery signature options does FedEx offer?

FedEx® Delivery Signature Options are as follows:

  • Deliver Without Signature. Authorizes FedEx to deliver your shipment without the recipient signing for it, and authorize a FedEx Express® shipment to be left without a signature at a nonresidential address. When you select this option, you waive your right to hold FedEx responsible for any resulting claims. (Not available for FedEx Express® Freight services.)
  • Direct Signature Required (DSR). FedEx obtains a signature from someone at the delivery address and reattempts delivery if no one is available to sign for the package. Available for residential and nonresidential deliveries.
  • Indirect Signature Required (ISR). FedEx obtains a signature in one of three ways:
    • From someone at the delivery address
    • From a neighbor, building manager or other person at a neighboring address
    • The recipient can leave a signed FedEx Door Tag authorizing release of the package without anyone present.

    Available for residential deliveries only.

  • Adult Signature Required (ASR). FedEx obtains a signature from someone at the delivery address who is at least 21 years old and who has government-issued photo identification as proof of age. FedEx reattempts delivery if no eligible recipient is available to sign for the package. Available for residential and nonresidential deliveries. Required for alcohol shipments.

For FedEx Express and FedEx Ground® shipments, the following fees apply for Indirect Signature Required, Direct Signature Required and Adult Signature Required services:

  • FedEx Express. Per package when the per-package rating applies; per shipment when the per shipment rating applies
  • FedEx Ground. Per package

Refer to the FedEx Service Guide for more information.

How does FedEx define a residential address?

A residential address is a home or private residence, including locations where a business is operated from the home.

What special services are available with U.S. shipments?

To display special services for FedEx Express®, FedEx Express® Freight, FedEx Ground® and FedEx Home Delivery® shipments, click the Options tab. FedEx Ship Manager displays only the special services that are available for your shipment.

  • Saturday Delivery. Deliver a package on Saturday if Saturday service is available for the shipment.
  • Delivery Signature Options. Select from four FedEx® Delivery Signature Options: Deliver Without Signature, Direct Signature Required, Indirect Signature Required, and Adult Signature Required. Alcohol shipments require an adult signature.


For FedEx Express Freight shipments, the following special services are available:

  • Inside Delivery. Deliver freight shipments inside the recipient’s building.

    • Has an actual weight of greater than 70 lbs.
    • Is not fully encased in an outer shipping container
    • Is encased in an outer shipping container made of metal or wood.

The surcharge also applies to any drum or pail (less than 5 gallons) that is not fully encased in an outer shipping container made of corrugated cardboard. The money-back guarantee does not apply to these shipments. We reserve the right to assess additional handling charges for packages that require special handling or that require FedEx to apply additional packaging during transit.

  • Delivery Signature Options. Select from two FedEx Delivery Signature Options: Direct Signature Required and Adult Signature Required.


For FedEx Home Delivery shipments, the following special services are available:

  • Appointment Delivery. Schedule delivery on the date and time that recipient indicates.
  • Date Certain Delivery. Schedule delivery on a certain date Tuesday through Saturday. Click the Delivery Date Calendar box and select the delivery date.
  • Evening Delivery. Schedule delivery between 5:00 p.m. and 8:00 p.m. in the recipient’s time zone.
  • Return Label. To print a return label, select this option and complete the Return Label screen.
  • Non-Standard Packaging. Check this option if your package:
    • Measures greater than 60 inches in length but equal to or less than 108 inches in length
    • Measures greater than 30 inches in length along its second-longest side
    • Has an actual weight of greater than 70 lbs.
    • Is not fully encased in an outer shipping container
    • Is encased in an outer shipping container made of metal or wood.

The surcharge also applies to any drum or pail (less than 5 gallons) that is not fully encased in an outer shipping container made of corrugated cardboard. The money-back guarantee does not apply to these shipments. We reserve the right to assess additional handling charges for packages that require special handling or that require FedEx to apply additional packaging during transit.

  • Delivery Signature Options. Select from four FedEx Delivery Signature Options: Deliver Without Signature, Direct Signature Required, Indirect Signature Required, and Adult Signature Required.
Can I ship a multiple-piece shipment using FedEx Ship Manager Software and FedEx Ship Manager hardware?

You can use FedEx Ship Manager Software and FedEx Ship Manager hardware to process a multiple-piece shipment using FedEx Express and FedEx Ground.

  • Return Labels. To print a return label, select this option and complete the Return Label screen.

Can I add handling charges to a multiple-piece shipment?

To add or modify handling charges for FedEx Express® or FedEx Ground® multiple-piece shipments, click Additional Handling and enter a fixed or variable amount. Handling charges carry over from one package to the next. To enable this option, follow these steps:

  1. Select Shipping Profiles from the Customize drop-down menu at the top of the screen.
  2. When the View Shipping Profiles screen displays, select a profile (e.g., Default Domestic Shipping Profile) and click View/Edit.
  3. When the View/Edit Domestic Shipping Profile screen displays, click the FedEx Express Preferences tab.
  4. Check the Include Additional Handling Charge option in the Handling Charge section.
  5. Enter a fixed or variable amount and click OK. Then click OK on the View Shipping Profiles screen. The additional handling charge option is now enabled.
Can I ship identical packages in a multiple-piece shipment?

If all of the packages in your shipment are exactly alike, select the Identical Packages checkbox and ship the packages using only one screen. If all packages are not exactly alike, do not select the Identical Packages checkbox.

What is a parse pak?

Parse pak is used to automatically divide the total weight of the shipment among all packages in a multiple-piece shipment. Parse pak is faster than entering information for each package. To use this option, simply enter the total number of packages and the total weight of the shipment to divide the weight equally among all packages. For Collect on Delivery (C.O.D.), enter one C.O.D. amount and receive one C.O.D. label for the shipment. You can only use parse pak with FedEx Ground® multiple-piece shipments.
